# Black Flag Funding

Black Flag Funding is a comprehensive platform for small business funding and loan solutions. It connects businesses with a nationwide network of over 80 underwriters to provide the best available financial deals.

## Key Features

- **Business Funding Matchmaking**: Access to 20+ lending programs including SBA loans, term loans, and lines of credit.
- **HeroCredit**: Business credit-building software to improve fundability without relying on personal credit.
- **Fast Approvals**: Streamlined process with funds deposited in as little as 24 hours for some programs.
- **Expert Guidance**: Personalized support from funding specialists to find the right path for your business.
- **Partner Program**: Referral opportunities for consultants and brokers with competitive commissions.

## Technology Stack

- **Framework**: [Next.js 15+](https://nextjs.org) (App Router)
- **Library**: [React 19](https://reactjs.org)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Forms**: [React Hook Form](https://react-hook-form.com) with [Zod](https://zod.dev) validation
- **Icons**: [Lucide React](https://lucide.dev)
- **HTTP Client**: [Axios](https://axios-http.com)
- **UI Components**: [Swiper](https://swiperjs.com) for carousels
- **Type Safety**: [TypeScript](https://www.typescriptlang.org)

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app`: Contains the pages, API routes, and layout.
- `src/components`: (Implied) Modular React components.
- `src/constants`: Configuration data for pages, metadata, and form fields.
- `src/types`: TypeScript interfaces and types.
- `public`: Static assets like images and fonts.
